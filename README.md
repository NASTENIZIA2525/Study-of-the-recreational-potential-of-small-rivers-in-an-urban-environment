# Исследование рекреационного потенциала малых рек в городской среде на примере реки Охты, г. Санкт-Петербург
## Аннотация. 
В работе предложена методика комплексной социально-экологической оценки (КСЭО) рекреационного потенциала прибрежных зон малых рек в городе. Основной принцип — раздельный расчёт двух характеристик: социальной востребованности участка (С) и его экологического состояния (Э). После этого для каждого участка определяется положение в матрице С/Э, что даёт типологию. Методика проверялась на пяти отрезках реки Охты (Санкт-Петербург) с заведомо разными условиями. Результаты показали, что один сегмент относится к типу «Инфраструктурный лидер», два — к «Территории комплексного кризиса», а два — выполняют роль экологических резервов. Инструмент позволяет ранжировать отрезки по очерёдности инвестиций и обосновывать сценарии развития не из общих соображений, а на конкретных цифрах. Материал ориентирован на использование в комитетах городского управления и в проектных организациях.
